## 业务描述

查看客户经理岗位员工的获豆结算明细数据。

财务人员或管理员通过账户月度信息的"结算明细"按钮，当员工岗位包含"经理"或"客户经理"关键词时，跳转到此页面查看对应的获豆明细，包括新客获豆、评价获豆、布草结余获豆、培训获豆等各类目明细。

---

## 业务流程

```
┌─────────────────────────────────────┐
│  从账户月度信息跳转（带岗位判断）     │
│  (month, employeeId, employeeName)   │
└──────────────┬──────────────────────┘
               │
               ▼
┌─────────────────────────────────────┐
│         筛选条件区域                 │
│  (月份、结算单号、时间、姓名...)     │
└──────────────┬──────────────────────┘
               │
               ▼
        [查询] 按钮
               │
               ▼
┌─────────────────────────────────────┐
│         数据表格区域                 │
│  (展示客户经理获豆明细数据)          │
└──────────────┬──────────────────────┘
               │
               ▼
         [导出] 按钮
```

---

## 筛选条件

| 字段 | 说明 | 控件类型 | 业务说明 |
|------|------|---------|---------|
| 月份 | 选择查询月份 | monthPicker | 选择查询的月份，格式YYYY-MM，默认从跳转参数带回 |
| 结算单号 | 输入结算单号 | input | 按结算单号精确查询某笔结算记录 |
| 结算发起时间 | 选择开始时间 | datetimePicker | 筛选发起时间范围起点 |
| 结算完成时间 | 选择结束时间 | datetimePicker | 筛选完成时间范围终点 |
| 姓名&工号 | 支持按姓名或工号搜索 | input | 输入员工姓名或工号进行搜索，支持模糊匹配，默认从跳转参数带回员工姓名 |
| 部门 | 输入部门名称 | input | 按部门维度筛选数据 |
| 岗位 | 输入岗位名称 | input | 按岗位维度筛选数据 |
| 结算状态 | 选择结算状态 | select | 可选：结算发起、结算成功、结算失败，用于筛选不同结算阶段的记录 |
| 主服务门店 | 主服务门店名称或编码 | input | 按主服务门店筛选 |
| 劳动关系门店 | 劳动关系门店名称或编码 | input | 按劳动关系门店筛选 |
| 合同主体 | 输入合同主体名称 | input | 按合同主体公司筛选 |
| 账户类型 | 固定选择"余额账户" | select | 当前固定为"余额账户" |

## 操作按钮

| 按钮 | 功能 | 详细说明 |
|------|------|---------|
| 查询 | 根据条件搜索 | 点击后根据筛选条件重新查询并展示客户经理获豆明细数据 |
| 重置 | 清空筛选条件 | 清空所有筛选条件输入框，恢复默认状态，表格数据重置为默认展示 |
| 导出 | 导出数据 | 导出搜索条件下所有符合条件的数据，导出字段包括：月份、结算单号、姓名、部门、岗位、工号、主服务门店、劳动关系门店、合同主体、新客获豆、评价获豆、布草结余获豆、消耗品结余获豆、资产维护获豆、住宿获豆、房间卫生打扫获豆、卫生间卫生打扫获豆、培训获豆、推拿师业绩达标获豆、保底获豆、试岗获豆、班次补贴获豆、三级合伙人获豆、客户经理人事手动补贴、获豆小计、失效常乐豆、本月获豆小计、结算数量、入账账户、结算发起时间、结算完成时间、结算状态，导出格式为 Excel |
