## 业务描述

按月份、员工信息筛选账户记录，查看账户汇总数据。

财务人员或管理员需要按月份查询员工的常乐豆账户汇总信息，包括结算、提现、消费等数据，支持按岗位跳转到不同的结算明细页面。

---

## 业务流程

```
┌─────────────────────────────────────┐
│         筛选条件区域                 │
│  (月份、姓名&工号、部门、岗位...)    │
└──────────────┬──────────────────────┘
               │
               ▼
        [查询] 按钮
               │
               ▼
┌─────────────────────────────────────┐
│         数据表格区域                 │
│  (展示账户月度汇总数据)              │
└──────────────┬──────────────────────┘
               │
         ┌─────┴─────┐
         ▼           ▼
    [结算明细]  [提现明细] [消费明细]
         │           │           │
         ▼           ▼           ▼
    按岗位跳转   常乐豆使用   常乐豆使用
    ┌────┴────┐   明细页     明细页
    ▼         ▼         ▼
[推拿师]  [客户经理]  [其他]

  明细      明细      明细
```

## 筛选条件

| 字段 | 说明 | 控件类型 | 业务说明 |
|------|------|---------|---------|
| 月份 | 选择查询月份 | monthPicker | 选择查询的月份，格式YYYY-MM，默认显示当前月份 |
| 姓名&工号 | 支持按姓名或工号模糊搜索 | input | 输入员工姓名或工号进行搜索，支持模糊匹配 |
| 部门 | 输入部门名称 | input | 按部门维度筛选账户数据 |
| 岗位 | 输入岗位名称 | input | 按岗位维度筛选账户数据，常用于筛选特定职位群体 |
| 主服务门店 | 主服务门店名称或编码 | input | 按主服务门店筛选，常用于门店人员数据查看 |
| 劳动关系门店 | 劳动关系门店名称或编码 | input | 按劳动关系门店筛选，用于区分门店与总部人员 |
| 合同主体 | 输入合同主体名称 | input | 按合同主体公司筛选，用于区分不同签约主体的员工 |
| 账户类型 | 固定选择"余额账户" | select | 当前固定为"余额账户"，用于筛选账户类型 |

## 表格列

| 列名 | 说明 | 业务说明 |
|------|------|---------|
| 月份 | 数据所属月份 | 格式为 YYYY-MM，用于标识数据统计周期 |
| 城市 | 员工所在城市 | 显示员工的工作城市区域 |
| 姓名 | 员工姓名 | 员工基本信息 |
| 部门 | 员工部门 | 员工所属组织部门 |
| 岗位 | 员工岗位 | 员工职位信息，根据岗位不同跳转到不同的明细页面 |
| 工号 | 员工工号 | 员工唯一标识 |
| 主服务门店 | 主服务门店名称 | 员工主要服务的门店，显示门店名称 |
| 劳动关系门店 | 劳动关系门店 | 员工签订劳动合同的门店或公司 |
| 合同主体 | 合同签约主体公司 | 员工签署劳动合同的公司主体 |
| 账户类型 | 账户类型标签 | 统一显示为"余额账户"，用标签标识 |
| 上月期末余额 | 上月结算后的账户余额 | 上个月结算周期结束时的账户余额 |
| 本月结算数量 | 本月新增的结算数量 | 本月通过业务结算获得的常乐豆数量（如服务收入、奖励等） |
| 本月提现数量 | 本月申请提现的数量 | 本月员工申请提现的常乐豆总数 |
| 本月累计消费 | 本月累计消费的常乐豆 | 本月通过消费（如兑换商品、服务）使用的常乐豆数量 |
| 提现交易成功金额 | 实际成功提现金额 | 本月提现申请中成功结算的金额（扣除失败部分） |
| 实时可用余额 | 当前可用余额 | 账户当前可用余额，高亮显示，用于实时了解账户状态 |

## 操作按钮

| 按钮 | 功能 | 详细说明 |
|------|------|---------|
| 查询 | 根据条件搜索 | 点击后根据筛选条件重新查询并展示账户月度汇总数据，支持按月份、员工信息等多维度筛选 |
| 重置 | 清空筛选条件 | 清空所有筛选条件输入框，恢复默认状态，表格数据重置为默认展示 |
| 导出 | 导出数据 | 导出搜索条件下所有符合条件的数据，导出字段包括：月份、城市、姓名、部门、岗位、工号、主服务门店、劳动关系门店、合同主体、账户类型、上月期末余额、本月结算数量、本月提现数量、本月累计消费、提现交易成功金额、实时可用余额，导出格式为 Excel |
| 结算明细 | 查看结算明细 | 根据员工岗位跳转到不同的结算明细页面：推拿师/理疗师/技师跳转到"推拿师获豆明细"；客户经理跳转到"客户经理获豆明细"；其他岗位跳转到"总部岗位获豆明细"。跳转时传递月份、员工ID、员工姓名参数 |
| 提现明细 | 查看提现明细 | 跳转到"常乐豆使用明细"页面，筛选条件默认为提现类型（type=withdrawal），跳转时传递月份、员工ID、员工姓名参数 |
| 消费明细 | 查看消费明细 | 跳转到"常乐豆使用明细"页面，筛选条件默认为消费类型（type=consumption），跳转时传递月份、员工ID、员工姓名参数 |

## 跳转参数

| 参数 | 来源 | 说明 |
|-----|------|------|
| month | URL Query | 月份，格式 YYYY-MM |
| employeeId | URL Query | 员工工号 |
| employeeName | URL Query | 员工姓名 |

## 分页说明

每页显示条数可选：10、20、50、100 条，支持跳转到指定页码。
