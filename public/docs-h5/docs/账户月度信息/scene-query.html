二 业务描述

按月份、员工信息筛选账户记录，查看账户汇总数据。

💡 说明

财务人员或管理员需要按月份查询员工的常乐豆账户汇总信息，包括结算、提现、消费等数据，支持按岗位跳转到不同的结算明细页面。

六 操作按钮

按钮功能详细说明
查询根据条件搜索点击后根据筛选条件重新查询并展示账户月度汇总数据，支持按月份、员工信息等多维度筛选
重置清空筛选条件清空所有筛选条件输入框，恢复默认状态，表格数据重置为默认展示
导出导出数据导出搜索条件下所有符合条件的数据，导出字段包括：月份、城市、姓名、部门、岗位、工号、主服务门店、劳动关系门店、合同主体、账户类型、上月期末余额、本月结算数量、本月提现数量、本月累计消费、提现交易成功金额、实时可用余额，导出格式为 Excel
结算明细查看结算明细根据员工岗位跳转到不同的结算明细页面：推拿师/理疗师/技师跳转到"推拿师获豆明细"；客户经理跳转到"客户经理获豆明细"；其他岗位跳转到"总部岗位获豆明细"。跳转时传递月份、员工ID、员工姓名参数
提现明细查看提现明细跳转到"常乐豆使用明细"页面，筛选条件默认为提现类型（type=withdrawal），跳转时传递月份、员工ID、员工姓名参数
消费明细查看消费明细跳转到"常乐豆使用明细"页面，筛选条件默认为消费类型（type=consumption），跳转时传递月份、员工ID、员工姓名参数

七 跳转参数

参数来源说明
monthURL Query月份，格式 YYYY-MM
employeeIdURL Query员工工号
employeeNameURL Query员工姓名

八 分页说明

每页显示条数可选：10、20、50、100 条，支持跳转到指定页码。
