## 跳转关系说明

### 结算明细跳转规则

| 岗位关键词 | 跳转目标页面 |
|-----------|-------------|
| 推拿师、理疗师、技师 | 推拿师获豆明细 |
| 经理、客户经理 | 客户经理获豆明细 |
| 其他岗位 | 总部岗位获豆明细 |

### 跳转参数

| 参数 | 来源 | 说明 |
|-----|------|------|
| month | URL Query | 月份，格式 YYYY-MM |
| employeeId | URL Query | 员工工号 |
| employeeName | URL Query | 员工姓名 |
| type | URL Query | 使用类型：withdrawal(提现)、consumption(消费) |

## 场景列表

1. **场景一：月度账户查询** - 按月份、员工信息筛选账户记录，查看账户汇总数据
2. **场景二：推拿师结算明细** - 查看推拿师/理疗师/技师岗位员工的获豆结算明细
3. **场景三：客户经理结算明细** - 查看客户经理岗位员工的获豆结算明细
4. **场景四：总部岗位结算明细** - 查看总部岗位员工的获豆结算明细
5. **场景五：提现明细** - 查看员工账户的提现记录和提现成功金额
6. **场景六：消费明细** - 查看员工账户的消费记录和累计消费金额
