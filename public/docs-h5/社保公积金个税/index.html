通用模式说明

📋 月度汇总页面模式
筛选条件（员工姓名/编号、月份）、统计卡片（本月缴纳合计、正常缴纳、补缴、退缴）、操作按钮（导出Excel、查看流水明细）
📝 月度流水明细页面模式
筛选条件（关键字、月份、记录类型、数据来源）、数据来源图例（区分三方服务/手动导入）、操作按钮（手动导入、新增记录、编辑）
🔄 跨月场景支持
费用产生月份、费用归属月份、费用实际缴纳月份三个字段用于处理社保公积金跨月补缴场景
