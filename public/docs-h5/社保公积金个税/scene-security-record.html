筛选条件

字段控件类型业务说明
关键字输入框支持按员工姓名或员工编号模糊搜索
月份月份选择器选择查询的月份，支持2026-01至2026-06等月份范围筛选
记录类型下拉选择可选：正常、补缴、退缴。用于区分正常缴纳和补退缴记录
数据来源下拉选择可选：三方服务、手动导入。用于筛选数据录入方式

表格列

列名业务说明
员工编号员工唯一标识，如E001
姓名员工真实姓名
等级员工等级，用于区分不同费率
费用类型包括「社保」（正常缴纳）和「滞纳金」（逾期未缴产生的罚款）
缴纳基数当月社保缴纳基数
缴纳金额当月实际缴纳金额
费用产生月份费用实际产生的月份（用于跨月补缴场景）
费用归属月份费用应归属的月份
费用实际缴纳月份费用实际缴纳到的月份（可能晚于归属月份）
类型正常/补缴/退缴，标识记录性质
来源三方服务/手动导入，标识数据来源
调整原因如为补缴或退缴，说明调整原因
记录时间记录创建时间，格式YYYY-MM-DD HH:mm:ss
操作「编辑」按钮，点击打开编辑弹窗

操作按钮

按钮功能详细说明
重置 清空筛选条件 1. 清空关键字输入框
2. 重置月份和记录类型为空
3. 数据来源恢复全部
4. 表格恢复展示全部数据
手动导入 批量导入记录 1. 打开导入弹窗
2. 支持拖拽或点击上传Excel文件（xlsx/xls/csv）
3. 选择导入月份和数据类型
4. 确认后系统提示导入成功
+ 新增记录 新增单条记录 1. 打开新增弹窗
2. 选择员工（员工编号自动关联姓名）
3. 填写月份、缴纳基数、缴纳金额
4. 选择记录类型（正常/补缴/退缴）
5. 选择数据来源（三方服务/手动导入）
6. 填写调整原因（如有）
7. 确认保存
编辑 编辑单条记录 1. 点击表格「编辑」按钮打开编辑弹窗
2. 表单预填充当前记录数据
3. 修改后确认保存
4. 自动更新员工姓名显示

弹窗说明

新增/编辑记录弹窗

字段控件类型说明
员工下拉选择选择员工，自动显示员工编号和姓名

缴纳基数数字输入输入缴纳基数，数值≥0

调整原因文本域选填，补充说明
