## 1. 页面概述

### 1.1 页面入口

```
推拿之家平台 → 推拿师 → 自主下线配置
```

### 1.2 页面说明

本页面用于配置推拿师自主下线的触发阈值，支持 Global > City > Store > Therapist 四级独立配置，并提供业务周日期明细生成功能。

## 2. 页面结构

### 2.1 整体布局

```
┌────────────────────────────────────────────────────────────────┐
│ 页面标题区域                                                    │
│ h1: 推拿师自主下线配置                                          │
│ 提示语: 💡 点击城市→门店→推拿师逐级配置阈值                    │
├────────────────────────────────────────────────────────────────┤
│ 全局默认配置卡片            │ 生成日期明细卡片                   │
│ [编辑按钮]                 │ [业务周下拉] [生成明细按钮]        │
│ 三项阈值展示               │                                    │
├────────────────────────────────────────────────────────────────┤
│ 城市列   │ 门店列          │ 推拿师列                           │
│ [+新增]  │ [城市标签][+新增]│ [门店标签][+新增]                 │
│ 列表项   │ 列表项          │ 列表项                             │
└────────────────────────────────────────────────────────────────┘
```

### 2.2 区域说明

| 区域 | 说明 |
|------|------|
| 全局默认配置卡片 | 展示兜底配置的三个阈值，点击编辑弹出修改弹窗 |
| 生成日期明细卡片 | 选择业务周后生成明细数据 |
| 城市列 | 一级配置，展示所有已配置城市 |
| 门店列 | 二级配置，选中城市后展示该城市下的门店 |
| 推拿师列 | 三级配置，选中门店后展示该门店下的推拿师 |

## 3. 功能详细说明

### 3.1 全局默认配置卡片

#### 3.1.1 展示内容

> **【需要截图：全局默认配置卡片，展开阈值展示状态】**

| 字段 | 说明 |
|------|------|
| 标题 | 🏠 全局默认配置 + [兜底配置] 标签 |
| 工作日阈值 | 显示当前值 + "分钟" 单位 |
| 周末阈值 | 显示当前值 + "分钟" 单位 |
| 节假日阈值 | 显示当前值 + "分钟" 单位 |

#### 3.1.2 操作

| 操作 | 说明 |
|------|------|
| 编辑 | 打开编辑弹窗，修改阈值 |

#### 3.1.3 编辑弹窗

> **【需要截图：编辑全局配置弹窗，三个数字输入框】**

**弹窗字段**

| 字段 | 类型 | 必填 | 校验规则 |
|------|------|------|----------|
| 工作日阈值 | 数字输入框 | 是 | >= 0，步进 30 |
| 周末阈值 | 数字输入框 | 是 | >= 0，步进 30 |
| 节假日阈值 | 数字输入框 | 是 | >= 0，步进 30 |

**按钮**：取消 / 保存

### 3.2 生成日期明细卡片

#### 3.2.1 展示内容

> **【需要截图：生成日期明细卡片】**

#### 3.3.1 列头

> **【需要截图：城市列列头 + 新增按钮】**

| 字段 | 说明 |
|------|------|
| 标题 | 🏢 城市 |

#### 3.3.2 列表项

> **【需要截图：城市列表项，展开阈值和操作按钮】**

| 字段 | 说明 |

| 门店数标签 | 绿色标签，显示该城市下的门店数量 |

**弹窗字段**

| 字段 | 类型 | 必填 | 说明 |

| 周末阈值 | 数字输入框 | 是 | 默认继承全局配置值，>=0，步进30 |
| 节假日阈值 | 数字输入框 | 是 | 默认继承全局配置值，>=0，步进30 |

#### 3.3.5 编辑弹窗

> **【需要截图：编辑城市弹窗，城市名称只读，阈值可编辑】**

**弹窗字段**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 城市 | 只读文本 | — | 显示城市名称，不可修改 |
| 工作日阈值 | 数字输入框 | 是 | >=0，步进30 |
| 周末阈值 | 数字输入框 | 是 | >=0，步进30 |
| 节假日阈值 | 数字输入框 | 是 | >=0，步进30 |

| 交互 | 说明 |
|------|------|

## 4. 接口设计

### 4.1 获取全局配置

**请求**

```
GET /api/offline-config/global
```

**响应**

```json
{
  "code": 0,
  "data": {
    "weekdayThreshold": 480,
    "weekendThreshold": 420,
    "holidayThreshold": 360
  }
}
```

---

### 4.2 更新全局配置

**请求**

```
PUT /api/offline-config/global
Content-Type: application/json

{
  "weekdayThreshold": 480,
  "weekendThreshold": 420,
  "holidayThreshold": 360
}
```

**响应**

```json
{
  "code": 0,
  "message": "保存成功"
}
```

---

### 4.3 获取城市列表

**请求**


**响应**

| 角色 | 权限 |
|------|------|
| 管理员 | 全部操作 |
| 人事经理 | 全部操作 |
| 其他 | 仅查看（无操作按钮） |

---

## 6. 异常处理

| 场景 | 提示 |
|------|------|
| 删除城市（存在门店配置） | 弹窗确认，告知将级联删除门店和推拿师配置 |
| 删除门店（存在推拿师配置） | 弹窗确认，告知将级联删除推拿师配置 |
| 新增时无可选对象 | 弹窗内下拉显示空，下拉置灰提示"暂无可选" |
| 保存失败 | 显示后端返回的错误 message |

## 7. 依赖说明

| 依赖项 | 说明 |
|--------|------|
| 业务周数据 | 下拉业务周列表依赖业务周配置数据 |
| 城市数据 | 城市列表依赖基础数据 |
| 门店数据 | 门店列表依赖基础数据 + 城市筛选 |
| 推拿师数据 | 推拿师列表依赖基础数据 + 门店筛选 |
