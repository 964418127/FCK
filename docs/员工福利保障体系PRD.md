# 员工福利保障体系 PRD

## 背景

原福利保障配置存在以下问题：
- 社保/公积金配置直接针对员工，无法关联岗位标准
- 缺乏城市薪资基准维度，无法标准化管理
- 无法追溯配置来源和变更历史

## 设计目标

建立"**标准→岗位→个人**"的三层福利保障体系：
1. **城市薪资基准** - 政策层面的缴纳标准
2. **岗位薪酬模板** - 引用基准，生成岗位标准
3. **员工福利明细** - 继承岗位标准，支持个体微调

## 数据结构

### 第一层：城市薪资基准（CitySalaryStandard）

政策/HR在系统中配置各城市的缴纳标准：

| 字段 | 说明 |
|------|------|
| 城市 | 如：北京、上海、深圳 |
| 薪资区间 | 如：3000-5000、5000-10000 |
| 缴纳类型 | 社保 / 公积金 |
| 缴纳基数 | 根据薪资区间确定 |
| 公司缴纳比例 | 法定比例 |
| 个人缴纳比例 | 法定比例 |

### 第二层：岗位薪酬模板（JobCompensationTemplate）

配置岗位时，通过岗位归属城市自动关联薪资区间，间接引用城市薪资基准。

薪酬模板中的**系统项**（社保、公积金）可配置为"自动获取"，无需手动填入具体数值。

### 第三层：员工福利明细（EmployeeWelfareDetail）

员工入职时：
1. 根据岗位 → 找到薪酬模板
2. 根据岗位的薪资区间 → 找到城市薪资基准
3. 自动生成员工福利明细（初始状态）

员工福利明细包含：
- 来源标准：标识继承自哪个城市薪资基准
- 实际配置：缴纳基数、比例
- Override标记：是否有微调

支持个体微调：
- 可调整：基数、比例
- 记录：调整人、时间、原因
- 保留历史变更日志

## 业务流程

```
[HR] 配置城市薪资基准
        ↓
[HR/管理者] 配置岗位薪酬模板（关联薪资区间）
        ↓
[系统] 员工入职时自动生成福利明细
        ↓
[员工] 需要调整 → 在明细中Override
        ↓
[系统] 薪酬计算时读取员工福利明细
```

## 页面清单

| 页面 | 功能 |
|------|------|
| 城市薪资基准配置 | 维护各城市+薪资区间的缴纳标准 |
| 员工福利保障 | 查看/调整员工福利明细（含Override标记） |
| 社保月度缴费明细 | 展示每月实际缴纳情况，含来源标记和手动导入 |
| 公积金月度缴费明细 | 展示每月实际缴纳情况，含来源标记和手动导入 |
| 个税月度缴纳明细 | 展示每月预扣情况，含汇算清缴、手动导入 |

## 月度缴费明细通用设计

### 数据来源

所有缴纳明细（社保/公积金/个税）都有两种数据来源：

| 来源 | 说明 | 场景 |
|------|------|------|
| **三方服务** | 百旺等系统自动推送 | 正常情况 |
| **手动导入** | 当三方未及时返回时人工录入 | 滞后/异常 |

### 每条记录的属性

```
数据来源: 三方服务 / 手动导入
缴纳类型: 正常 / 补缴 / 退缴
状态: 已确认 / 待确认
```

### 个税特殊情况

个税包含额外的类型：
- 正常（每月预扣）
- 补税（汇算清缴需补）
- 退税（汇算清缴可退）
- 汇算清缴（年度汇算）

### 1. 城市薪资基准的优先级

当城市薪资基准发生变更时：
- **已入职员工**：不受影响（已生成独立明细）
- **新入职员工**：使用最新基准

### 2. 薪酬计算时的数据获取

```
薪酬计算 → 获取员工ID → 查询员工福利明细
                               ↓
                    有Override → 使用员工配置的値
                    无Override → 使用城市薪资基准
```

### 3. 个体微调的审计

每次Override记录：
- 时间、操作人
- 调整字段、原值、新值
- 调整原因（员工申请/公司政策/薪资调整/其他）

## 与其他模块的关系

```
城市薪资基准 ──引用──→ 岗位薪酬模板（系统项配置）
                              │
                              ↓
薪酬发放策略 ← 关联 ────── 员工福利明细（最终计算依据）
```

## 页面原型

见各页面 Vue 组件：
- `CitySalaryStandard.vue` - 城市薪资基准配置
- `EmployeeWelfareDetail.vue` - 员工福利保障明细
